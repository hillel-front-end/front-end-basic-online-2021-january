Важно знать:

Margin

margin: 0 auto; - при этом если дочерни блок меньше родительского блока, то дочерний блок- будетотцентрирован по горизонатали. Для вертикали margin-top,margin-bottom "auto = 0".(Блочные)

Display

Свойство CSS Display нужен для установки типа html элемента относительно других элементов. Проще говоря это опция отвечает за то, как отображать элемент относительно други. Значения: *block - блочный элемент (автоматически создает перевод строки) *inline - встроенный элемент (не создает перевода строки, и не доступны сss cвойства: width\height, margin-top, margin-bottom); *none - Элемент не показывается, вообще. Как будто его и нет.

inline-block

Это значение – означает элемент, который продолжает находиться в строке (inline), при этом может иметь важные свойства блока. Как и инлайн-элемент(inline): - Располагается в строке. - Размер устанавливается по содержимому. Во всём остальном – это блок, то есть: - Работают свойства width/height. - доступны margin-top, margin-bottom -->
